## Power Input and Rails:

## Battery Input
`6-12V Battery Input for servos and motors offboard` <br>
`Battery input with TVS, bulk capacitor, polyfuse and reverse polarity protection` <br>
`->Battery to 5V rail via buck converter` <br>
`5v rail from ESC to FC_5V, through Schottky diode` <br>
`->FC_5V to 3.3v rail via LDO for sensors/MCU` <br>

## USB Input
`5v VBUS input to FC_5V rail, via Schottky diode` <br>
`TVS protection on DM and DP lines, as well as VBUS`
